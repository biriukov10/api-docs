Private HTTP API V1

Private endpoints V1‚Äã

Those endpoints are deprecated use V4 alternatives‚Äã

  • Trading balance by currency
  • Trading balances
  • Create limit order
  • Cancel order
  • Query unexecuted orders
  • Query order history
  • Query order history by single market
  • Query order history by all markets
  • Query executed order deals

Base URL is https://whitebit.com

Endpoint example: https://whitebit.com/api/v1/{endpoint}

All endpoints return time in Unix-time format.

All endpoints return either a JSON object or array.

For receiving responses from API calls please use http method POST

Error messages V1 format:‚Äã

{
    "code": 0,
    "message": "MESSAGE",
    "errors": {
        "PARAM1": [
            "MESSAGE"
        ],
        "PARAM2": [
            "MESSAGE"
        ]
    }
}

Terminology‚Äã

Pair:‚Äã

Stock - currency that you want to buy or sell

Money - currency that you are using to buy or sell something

Maker - person who puts an order and waiting till this order will be finished

Taker - person who finishes existing order

Precision - is the number of digits to the right of the decimal point

Bid - buy order

Ask - sell order

Limit order - to place this order, you need to fill the 'Price' and 'Amount' fields. If this order finds a corresponding order on the opposite side, it will be executed. Otherwise it will be placed into the orderbook.

amountStringYesAmount of stock currency to buy or sell. Example: '0.001'
priceStringYesPrice in money currency. Example: '9800'
clientOrderIdStringNoIdentifier should be unique and contain letters, dashes or numbers only. The identifier must be unique for the next 24 hours.
